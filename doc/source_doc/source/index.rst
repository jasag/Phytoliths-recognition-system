**Bienvenidos a la documentación de Phytoliths-recognition-system !**
=====================================================================

Phytoliths-recognition-system es un proyecto en el que desarrollamos
un sistema de reconocimiento automático de fitolitos y un etiquetador
de los mismos. Generando distintas herramientas y aplicando múltiples
técnicas de procesamiento de imágenes y de inteligencia artifial.

La mayoría de las herramientas están documentadas y creadas en Jupyter Notebooks,
pero existen algunos modulos que se encuentran empaquetados en modulos, los cuales
deben ir adecuadamente documentados.

**Prerequisitos**:

- Windows.
- Google Chrome.
- Anaconda.
- Ipython File Upload
- Jupyter Dashboards
- Algunos otros, que más tarde especificaré.
- La ultima release de este proyecto o repositorio.
